## ツール

### pgAdmin – GUI クライアント

[pgAdmin](https://www.pgadmin.org) は、PostgreSQL のための GUI クライアントツールで、クロスプラットフォーム上で動作し、日本語に対応しています。

#### インストール

- [インストーラ](https://www.pgadmin.org/download/)
- Homebrew

  ```bash
  $ brew install --cask pgadmin4
  ```
