# フィード

## フィードの種類

### YouTube チャンネル

```
チャンネルIDから取得： 
https://www.youtube.com/feeds/videos.xml?channel_id={チャンネルID}

ユーザー名から取得：
https://www.youtube.com/feeds/videos.xml?user={ユーザー名}
```
