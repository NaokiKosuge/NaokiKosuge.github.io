VSCode for Mac キーボードショートカット

情報源
  • Visual Studio Code Keyboard shortcuts for macOS

一般​

ショートカット説明
⇧⌘P, F1コマンドパレットを表示する
⌘Pクイックオープン、ファイルに移動
⇧⌘N新しいウィンドウ／インスタンス
⌘Wウィンドウ／インスタンスを閉じる
⌘,ユーザー設定
⌘K ⌘Sキーボードショートカット

基本的な編集​

ショートカット説明
⌘Xカットライン（空の選択）
⌘C行をコピー（空の選択）
⌥↓ / ⌥↑行を下/上に移動する
⇧⌥↓ / ⇧⌥↑行を下/上にコピーする
⇧⌘K行を削除する
⌘return / ⇧⌘return行を下/上に挿入する
⇧⌘\大括弧にジャンプする
⌘] / ⌘[インデント/アウトライン
Home / End行の先頭/末尾に移動する
⌘↑ / ⌘↓ファイルの先頭/最後に移動する
⌃PgUp / ⌃PgDnスクロールアップ/ダウン
⌘PgUp / ⌘PgDnページを上下にスクロールする
⌥⌘[ / ⌥⌘]リージョンを折り畳む/展開する
⌘K ⌘[ / ⌘K ⌘]全てのサブリージョンを折り畳む/展開する
⌘K ⌘0 / ⌘K ⌘J全てのリージョンを折り畳む/展開する
⌘K ⌘C行コメントを追加する
⌘K ⌘U行コメントを削除する
⌘/行コメントを切り替える
⇧⌥Aブロックコメントを切り替える
⌥Z単語の折り返しを切り替える

マルチカーソルと選択​

ショートカット説明
⌥ + clickカーソルを挿入
⌥⌘↑カーソルを上に挿入する
⌥⌘↓カーソルを下に挿入する
⌘U最後のカーソル操作を元に戻す
⇧⌥Iカーソルを選択した各行の最後に挿入する
⌘L現在の行を選択する
⇧⌘L現在の選択のすべてのオカレンスを選択する
⌘F2現在の単語のすべてのオカレンスを選択する
⌃⇧⌘→ / ←選択範囲を拡大/縮小
⇧⌥ + drag mouse列（ボックス）の選択
⇧⌥⌘↑ / ↓列（ボックス）の上/下選択
⇧⌥⌘← / →列（ボックス）の左/右選択
⇧⌥⌘PgUp列（ボックス）ページアップ選択
⇧⌥⌘PgDn列（ボックス）ページダウン選択

検索と置換​

ショートカット説明
⌘F検索
⌥⌘F置換
⌘G / ⇧⌘G次/前を検索
⌥return一致するものをすべて選択
⌘D次の検索一致に選択を追加
⌘K ⌘D最後の選択を次の一致の検索に移動

豊富な言語編集​

ショートカット説明
⌃space, ⌘Iトリガーの提案
⇧⌘spaceトリガーパラメーターのヒント
⇧⌥Fドキュメントのフォーマット
⌘K ⌘F選択範囲のフォーマット
F12定義へ移動
⌥F12定義をここに表示
⌘K F12定義を横に開く
⌘.クイックフィックス
⇧F12参照へ移動
F2シンボルの名前変更
⌘K ⌘X末尾の空白のトリミング
⌘K M言語モードの変更

ナビゲーション​

ショートカット説明
⌘Tワークスペース内のシンボルへ移動
⌃G行/列に移動
⌘Pファイルに移動
⇧⌘Oエディターでシンボルに移動
⇧⌘M問題パネルを表示
F8 / ⇧F8ファイル内の次/前の問題（エラー、警告、情報）へ移動
⌃⇧tabエディターグループの履歴をナビゲート
⌃- / ⌃⇧-戻る/進む
⌃⇧Mトグルタブがフォーカスを移動

編集管理​

ショートカット説明
⌘Wエディターを閉じる
⌘K Fフォルダを閉じる
⌘\エディターの分割
⌘1 / ⌘2 / ⌘31番目/2番目/3番目のエディターグループにフォーカスする
⌘K ⌘← / ⌘K ⌘→前/次のエディターグループにフォーカス
⌘K ⇧⌘← / ⌘K ⇧⌘→エディター左/右に移動
⌘K ← / ⌘K →エディターグループを左/右に移動

ファイル管理​

ショートカット説明
⌘N新規ファイル
⌘Oファイルを開く
⌘S保存
⇧⌘S名前を付けて保存
⌥⌘Sすべて保存
⌘W閉じる
⌘K ⌘Wすべて閉じる
⇧⌘T閉じたエディターを再度開く
⌘K returnエディターを保持
⌃tab / ⌃⇧tab次/前を開く
⌘K Pアクティブファイルのパスをコピー
⌘K RFinder でアクティブファイルを表示
⌘K O新しいウィンドウ/インスタンスでアクティブなファイルを表示

ディスプレイ​

ショートカット説明
⌃⌘F全画面表示の切り替え
⌥⌘0エディターレイアウト（水平/垂直）の切り替え
⌘= / ⇧⌘-ズームイン/アウト
⌘Bサイドバーの表示の切り替え
⇧⌘Eエクスプローラーを表示/フォーカスの切り替え
⇧⌘F検索を表示
⌃⇧Gソース管理を表示
⇧⌘D実行とデバッグを表示
⇧⌘X拡張機能を表示
⇧⌘H複数のファイルで置換
⇧⌘J検索の詳細の切り替え
⇧⌘U出力の切り替え
⇧⌘Vマークダウンプレビューを開く
⌘K Vマークダウンプレビューを横に表示
⌘K ZZen モードの切り替え（esc esc で終了）

デバッグ​

ショートカット説明
F9ブレークポイントの切り替え
F5開始/続行
F11 / ⇧F11ステップイン/アウト
F10ステップオーバー
⇧F5停止
⌘K ⌘Iホバーの表示

統合ターミナル​

ショートカット説明
⌃`ターミナルの切り替え
⌃⇧`新しい統合ターミナルの作成
⌘C選択内容のコピー
⌘↑ / ↓スクロールアップ/ダウン
PgUp / PgDnスクロールページアップ/ダウン
⌘Home / End一番上/下にスクロール
